MyFlix Documentation

This application delivers information about various movies, directors, and genres. Users can sign up, update their contact details, and build a personalized list of favorite movies.

Request URL HTTP Method Request Body Data Format Response Body Datat Format
Get a list of all movies /movies GET none A JSON object showing data of all movies
Get data about a single movie by title /movies/[title] GET none A JSON object data about a single movie containing the description, genre, director, image URL. Example: 
                            {
                                title:'Death Becomes Her',
                                Description: '',
                                Genre: 'Comedy, Fantasy'
                                Director: 'Robert Zemeckis',
                                Image: ''
                            },
Get data about a genre /movies/genres/[genre] GET none A JSON object showing the details of a genre. Example: 
                            {
                                genre: 'Comedy',
                                description: ''
                                
                            },
Get data about a director by name /movies/directors/[director] GET none A JSON object showing data of director containing bio, birth year. Example: 
                            {
                                director: 'Robert Zemeckis',
                                bio:'',
                                birth year: ''
                            },
Allow a new user to register /users POST A JSON object showing data about the new user. 
                            {
                                name: '',
                                username: ''
                            },
A JSON object showing their information. Example: 
                                {
                                    name: '',
                                    username: ''
                                    id: ''
                                },
Allow users to update their user info (username) /users/[id] PUT A JSON object with the date to be updated. 
                            {
                                username: ''
                            },
A message indicating that the username was updated sucessfully or an error message if not.
Allow users to add a movie to their list of favorites /users/[id]/[favorites] POST A JSON object with the movie to be added. 
                                {
                                    title:'Death Becomes Her',
                                },
A message indicating that the movie was added sucessfully or an error message if not.
Allow users to remove a movie from their list of favorites /users/[id]/[favorites] DELETE none A message indicating that the movie has been removed from favorites successfully or an error message if not.
Allow existing users to deregister /users/[id] DELETE a JSON object containing the user to be removed. 
                                {
                                    name: ''
                                    username: ''
                                },
A message indicating that username has been removed or an error message if not.
